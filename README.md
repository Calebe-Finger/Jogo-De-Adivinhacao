# Jogo de Adivinhação

![](https://imgur.com/xFsSGMg.gif)

## Introdução 

Trata-se de um jogo em que a máquina gera um número aleatório, o qual o usuário
precisa adivinhar com base nas dicas da máquina.

## Funcionaçidades

- **Escolha de dificuldade:** fácil, médio e difícil
> Quanto maior a dificuldade, menor o número de tentativas; 10, 5 e 3, respectivamente.

- Validação do número inserido
- Exibe o número de tentativas utilizadas e o número restante.
- Exibe a pontuação atual do usuáario.
- Armazena e exibe os números já chutados pelo usuário
- Não possibilita o usuário inserir o mesmo número mais de uma vez.
- Exibe dicas após a tentativa do usuário: 
> O número digitado é maior(menor) que o número secreto.

- Quando o usuário acerta o número secreto, exibe a pontuação final.
- Possibilita o usuário jogar várias vezes seguidas.
